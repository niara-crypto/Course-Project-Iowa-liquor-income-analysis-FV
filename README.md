# Iowa Liquor vs Income (2012\'962016)\
\
**Research Question:** How does median household income correlate with per-capita liquor sales across Iowa counties (2012\'962016)?\
\
## How to reproduce\
1. Open R and run `r/merge_counties.R` (it outputs `data/merged_county_data_for_tableau.csv`).\
2. Open Tableau workbook in `tableau/scatter_income_vs_percap.twbx`.\
\
## Key result\
- Trend: Negative slope\
- Correlation (R): r \uc0\u8776  -0.106 (weak negative)\
\
